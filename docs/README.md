# 📚 CareerOS Documentation

Bienvenido a la documentación oficial de **CareerOS**.

CareerOS es mi sistema personal para gestionar mi crecimiento académico y profesional durante la carrera de Ingeniería de Sistemas.

## Objetivos

- Documentar toda mi carrera universitaria.
- Llevar el seguimiento de materias y créditos.
- Construir un portafolio profesional.
- Obtener certificaciones alineadas con mis objetivos.
- Prepararme para trabajar como Software Engineer.

## Estructura

- Dashboard
- Perfil
- Universidad
- Roadmap
- Tecnologías
- Certificaciones
- Proyectos
- Empleabilidad
- Recursos
- Historial de cambios
